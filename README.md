# DPLL
## Description
DPLL based SAT Solver
and
CDCL based SAT Solver
## Usage
    python3 dpll.py <file path>
or

    python3 cdcl.py <file path>

It also supports input from console by leaving
<file path\> empty.
## Input Format
DIMACS-CNF Format
## Course
Logic and Program Verification
